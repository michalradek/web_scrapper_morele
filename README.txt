# Morele Web Scrapper

# Description

A simple web scraper for the Morele.net store. The program allows you to scrape the price of any item from Morele.net.

#Tech stack

The application is written in Python. The frontend is built using the Tkinter library. The backend uses the following libraries:

- sqlite3 (for database operations)
- requests (for sending GET requests)
- re (for input sanitization)
- BeautifulSoup (for web scraping)
